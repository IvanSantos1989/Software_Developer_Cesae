ToDoAppSwiftUI - Fonte SwiftUI (Swift)
--------------------------------------
Estes arquivos são para você colar em um projeto iOS (SwiftUI) no Xcode.

Passos rápidos (Xcode):
1) File > New > Project… > iOS > App (SwiftUI, Swift).
2) No navegador do projeto (Project Navigator), apague os arquivos ContentView.swift e <AppName>App.swift criados automaticamente.
3) Arraste os arquivos 'ContentView.swift' para dentro do projeto (se preferir, divida em arquivos depois).
4) Execute no simulador (⌘R).

Observação: Este ZIP contém os fontes prontos. Caso prefira um .xcodeproj completo,
crie o projeto vazio e substitua os arquivos conforme indicado.
